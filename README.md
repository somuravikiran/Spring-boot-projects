School Application 🚀
Full-Stack School Management System built with Spring Boot 4.0.1, Thymeleaf, H2 Database, and Spring Security.

✨ Features
Contact Form - User inquiry submission with validation

Admin Messages Dashboard - View & manage open contact messages

Responsive UI - Bootstrap 5 + custom gradient buttons

Role-based Security - ADMIN vs USER access control

RESTful APIs - Clean MVC architecture

H2 In-Memory Database - Quick development & testing
